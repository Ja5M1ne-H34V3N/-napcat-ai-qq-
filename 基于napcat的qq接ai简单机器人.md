练手玩的：
目前只有很简单的功能，自行配置api模型，并且部署在本地，同时可以从qq私信的方式来完成一些操作
目前只支持linux部署，自行配置api模型
# 部署方法：
在进行程序的克隆之后，根据requirements下好需要的外部库，打开目录下bot文件夹中的main.py，配置第一部分llm的ChatOpenAI的模型
```pyhton
llm = ChatOpenAI(

    base_url="", #自行设置模型api

    api_key="",

    model=""

)
```

接下来，尝试运行main.py，第一次运行，需要程序自动下载napcat，输入y即可，接下来，控制台会弹出一个二维码，手机qq扫码登录即可，如果程序没有报错，则说明部署成功，后续的使用操作详见napcat官方手册和下面的功能

第一次及之后的运行，都会提示你输入主人qq，在未来，只有收到这个qq号发来的消息才会有回复

# 功能：
目前本机器人支持的指令格式为：/指令
仅支持三种指令；
'/aimode', 进入ai对话模式
'/quitaimode'，退出ai对话模式
'/help'，展示指令列表
进入ai对话模式后，可通过主人qq向机器人qq发送消息来进行操作，目前非ai对话模式，无论输入什么非指令内容，都统一回复ccb
